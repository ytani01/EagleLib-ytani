# EagleLib-ytani

EAGLE Libraries

## install

### 1. clone git repository

```bash
$ git clone https://www.github.com/ytani01/EagleLib.git
or 
$ git clone git@github.com:ytani01/EagleLib.git
```

### 2. copy *.lbr files

Distination Folder
```
C:\Users\{username}\Documents\EAGLE\libraries\EagleLib-ytani
```
